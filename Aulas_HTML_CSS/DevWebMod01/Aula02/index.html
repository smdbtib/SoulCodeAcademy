CSS inline

Esse método é o que utilizamos na construção dos nossos códigos, até o momento. Trata-se de inserir o comando CSS dentro da linha em que você precisa configurar, através do atributo style, que fica dentro da tag de abertura de um elemento.

CSS externo

O CSS externo é a forma mais utilizada pelos programadores de maneira geral, e consiste em separar totalmente o código de configuração, do código de estrutura da página. Os comandos de configuração ficam em um arquivo separado, salvo com a extensão .css, e as páginas HTML recebem apenas um link, dentro do , para chamar esse arquivo.
